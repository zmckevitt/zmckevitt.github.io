Automatic Transient Execution Attack Detection

Overview

Building a probabilistic learning platform to identify suspicious code execution patterns that indicate Speculative Execution (Spectre) Attacks. Our platform attempts to follow program flow at the microarchitectural level to observe hardware state changes on a simulated x86 architecture with the gem5 simulator. We plan to use Probabilistic Programming Languages (PPLs) to identify these patterns by training the algorithm with Proof of Concept (PoC) Spectre implementations. Once a sufficient training set has been collected, our PPL algorithm will be able to learn a finite state machine that is able to detect suspicious program flows in arbitrary programs.

This research has been presented at the University of Colorado Undergraduate Research Symposium as well as the HASP workshop at MICRO '21. This is also the focus of my Senior Thesis.
