zackmckevitt.com

With a raspberry pi, an apache web server, a domain, and some port forwarding, I hosted this website on my home network. This project was for me to build exposure to basic networking and gain proficiency with the command line. With the convinving of my roommates, I was able to set up a DHCP reservation for my rasbperry pi so I wouldnt have to repoint the domain every week, and I opened port 80 for my server. After purchasing the zackmckevitt.com domian, I set up a basic DNS to point my A record domain at my pi.
