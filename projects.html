Selected Class Projects

Distributed File System - CSCI 5273

C

Built a distributed file system for CSCI 5273 in C using sockets for Linux based systems. Users were able to store and retrieve files on a de-centralized server (comprised of 4 servers each on a unique port) by uploading pieces of the file to each server. This program implements redundancy in the case of a single server failure, user authentication, and simple client side encryption that ensures the file pieces are encrypted both in transit to and from the server and when stored on the servers. The github for this project can be found here.

HTTP Proxy Server - CSCI 5273

C

Built a browser-usable proxy server for websites using HTTP in C. This simple proxy was able to cache resources for a user defined length of time, cache DNS results, blacklist domain names or IP addresses, and pre-fetch resources for pages referenced via hyperlinks. The server also implemented multithreading to ensure that simultaneous connections would be handled.

Multithreaded DNS Resolver - CSCI 3573

C

Wrote a DNS resolver capable of using multiple threads in a thread-safe manner. A fixed-length buffer was used to store domain names with multiple threads adding to and pulling from said buffer. Synchronization techniques such as mutex locks and condition variables were used to eliminate race conditions from the buffer, and the program's use of multiple threads ensured a faster run-time than that of a single threaded program.

File I/O Driver LKM For Linux - CSCI 3573

C

Wrote a basic driver for file I/O as a LKM to be loaded into the Linux Kernel. This program achieved the same functionality as library functions such as fwrite() and fread() and directly manipulated inode pointers. This LKM was written in kernel space and we were responsible for successfully loading our LKM into the Linux kernal and using it.
